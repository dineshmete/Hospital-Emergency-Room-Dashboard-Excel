# Hospital Emergency Room Dashboard using Excel

## Project Overview

This project presents an interactive Hospital Emergency Room Dashboard developed using Microsoft Excel to analyze hospital operations and improve healthcare efficiency through data-driven insights.

The dashboard helps monitor:
- Patient visits
- Average waiting time
- Patient satisfaction score
- Admission status
- Department referrals
- Gender distribution
- Age-group analysis

The project was created to help hospital management improve operational efficiency, monitor emergency room performance, and support better decision-making.

# Business Problem

Hospitals require efficient reporting systems to:
- Reduce patient waiting time
- Improve patient satisfaction
- Analyze emergency room performance
- Track department referrals
- Support operational decision-making

This dashboard transforms raw hospital data into meaningful insights using Excel dashboards and analytics techniques.

---

# Project Objectives

- Analyze emergency room patient trends
- Monitor average patient waiting time
- Track admission vs non-admission rates
- Measure patient satisfaction levels
- Identify high-referral departments
- Improve operational efficiency using analytics

| KPI | Description |

# Project Workflow

## 1. Business Requirement Gathering
- Identified healthcare reporting requirements
- Defined KPIs and dashboard objectives

## 2. Data Understanding
- Analyzed hospital patient datasets
- Studied data attributes and relationships

## 3. Data Connection
- Imported and connected raw hospital data in Excel

## 4. Data Cleaning & Quality Check
Performed:
- Null value handling
- Duplicate removal
- Data validation
- Format standardization
- Data quality checks

## 5. Calendar Table Creation
- Created Date/Calendar table for monthly and yearly analysis

## 6. Data Modeling
- Built relationships between tables using Power Pivot
- Structured data model for reporting and analytics

## 7. DAX Calculations
Created:
- Calculated columns
- Measures
- KPI metrics using DAX

## 8. Dashboard Development
Designed:
- KPI Cards
- Pivot Tables
- Pivot Charts
- Slicers
- Interactive Reports

## 9. Insight Generation
Generated insights related to:
- Patient flow trends
- Wait time analysis
